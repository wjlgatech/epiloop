# `epiloop pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
epiloop pairing list whatsapp
epiloop pairing approve whatsapp <code> --notify
```
