# `epiloop tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
epiloop tui
epiloop tui --url ws://127.0.0.1:18789 --token <token>
epiloop tui --session main --deliver
```
